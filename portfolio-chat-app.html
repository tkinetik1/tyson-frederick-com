Friendly Chat with Firebase

Friendly Chat is a Firebase Android chat application. It is simple, quick, modern, minimal, and up to date with Google's current Material Design and Android development standards. This application is written in conjunction with version 1 and version 2 of Udacity's: Developing Android Applications. This course is meant to cover all of the fundamentals and building blocks of Android Development as well as their implementation of java and xml. For those of you looking to learn all about the fundamentals of Android Application Development, taught by Google's professional advocates, please visit the Udacity Course Catalog.
